    Card 卡片

    # 基础使用

    基础卡片包含标题、内容等部分。

    # 封面

    可以使用 cover 属性设置封面。

    # 属性列表

    属性 说明 类型 默认值
    bodyStyle 卡片内容区内联样式 CSSProperties
    bordered 是否设置卡片的外边框 boolean
    footerLine 卡片页脚区与内容区是否有边线 boolean
    footerStyle 卡片页脚区内联样式 CSSProperties
    headerLine 卡片标题区与内容区是否有边线 boolean
    headerStyle 卡片标题区内联样式 CSSProperties
    shadows 设置显示阴影的时机，如果不设置该属性则没有阴影，可选值：hover、always string
    title 卡片标题 string

    # 插槽

    属性 说明 类型 默认值
    header 自定义卡片头部 node
    cover 卡片封面 node
    footer 自定义卡片页脚 node
